# 2072 代号→中文名对照表 v1

**用途：** 全书改写时正文一律使用"中文意象名"列。代号系统作为附录索引保留，正文首次出现时以括注形式给出对应代号一次（如"美式路径（A1）"），其后只用中文。
**命名原则：** (1) 意象优先于直译；(2) 与现有政经术语保持半步距离；(3) 一节内不出现两个易混名；(4) 避开已被特定政治叙述占用的词汇。
**待用户裁定列：** 命名为多个候选时已用"／"列出，按推荐度从高到低排列。

## 三　主因果链 A—F

主因果链 A—F 在文件中尚未在正文给出完整命名，仅在速览页提及"六条主因果链"。建议在改写时正式命名为：

| 代号 | **建议中文名** | 主要内容 |
|------|---------------|---------|
| A | **劳动塌陷链** | 替代率上升 → 劳动收入份额下降 → 分配逻辑失基础 |
| B | **流动收缩链** | 离乡回报下降 → 人口回流本地 → 社区粘度上升 |
| C | **意义真空链** | 劳动从分配中退场 → 意义供给来源转移 → 需依赖社区与文化机制承接 |
| D | **国家分叉链** | 国家手伸长度差异 → 制度响应路径不同 → 美式与中国两大坐标分形 |
| E | **痛苦稳压链** | 痛苦政治化温度受物质兜底、缺乏集体目标、动荡记忆三重压制 → 高痛苦不自动转高动荡 |
| F | **历史落差链** | 同一冲击在不同路径上消化时间不同 → 同一物理年份各地处于不同"历史阶段" |

正文调用时使用"……此即劳动塌陷链所揭"或"详见劳动塌陷链一节"。

## 九　人物身份的类型化命名规范

不出现具名个体（红线一）。当需要"一个人"作为论述支点时，统一格式：

> **`地理 + 时段 + 职业／身份`**

样例：

- "2052 年青海某县的一位票据退休人员"
- "2068 年皮尔巴拉的一位合成油站值班员"
- "2045 年成都某区的一位 K12 数学教师"
- "2058 年加州一位积分托底制下的中年失业建筑师"
- "2070 年印度班加罗尔一位维持双轨技术栈的工程师"

**禁用：** 任何具名（陈某、谢某等具名个体只属于场景项目和小说项目）。
**允许：** 同一类型多次出现，不必声明"还是同一个人"——它本来就不是一个人。

## 附：使用流程

每改写一节，作者：
1. 检查本节出现的代号；
2. 全部替换为本表"建议中文意象名"列；
3. 节内首次出现时附括注（如"美式路径（A1）"）；
4. 该节其后不再用代号；
5. 若发现本表未覆盖的新代号或缩写，补充本表后再改写。
