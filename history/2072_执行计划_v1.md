## 一、总则

**工程目标：** 把 v143（28.7 万字理论推演文件）改写为 v144（约 67 万字六卷可读著作）。
**预计工期：** 5—7 个月。**任务单位：** 章（chapter）。**总章数：** 约 80 个。
**核心约束：**
- 一次会话上下文有限，须以章为最小独立单位写作；
- 工具集（文风手册 / 代号对照表 / 史鉴对照表）已稳定，写作时按规调用；
- 任何阻塞或裁定问题写入进度看板，不自行拍板；
- 已完成的三份示范节为基线（1.4.1 / 3.14.6 / 6.7.5）。

**作者已裁定（见 2026-05-08 决策记录）：**
1. 卷 3 制度分化采用纵切（每条路径独立成章）；跨路径对比放卷 3 章十二之前的"路径对比"专章。
2. 卷 4 把动荡期溶解进时间轴。
3. 数据与媒体按草案：部分入路径章、部分入卷 3 章九。
4. 锚点年切片两版互文：卷 0 厚版（每年 1500 字）、卷 4 简版（每年 800 字）。
5. V1—V8、A—F 代号部分下沉：正文用中文名，章末附对照表。
6. 5.6.2 民主东亚归卷 3 章四盟邦路径。
7. **5.7.4 资源出口国分拆：卷 3 讲过程（制度分化与历史响应），卷 5 讲结果（地理形态与人口形态）**——本条偏离草案，需在执行时双卷协同。
8. 4.13 国际金融按草案归卷 3 章七。
9. 卷 0 路径速览与卷 3 路径总图各司其职：卷 0 是 5 分钟轮廓，卷 3 是该路径全图。

**Phase 顺序为什么这样排：**
- Phase 1 先做"跨路径对比章设计"（不是写）和"5.7.4 拆分文档"，这两件事是后续 Phase 5、Phase 7 的依赖，必须先到位。
- Phase 2（卷 0）先做，是因为卷 0 的锚点年切片是后续所有时间线工作的"皮"。卷 0 完成后，作者读完前 30 页就能验证整个新书的方向。
- Phase 3（卷 2 理论）次之，是因为理论骨架定下来后，卷 1 物质基础、卷 3 制度分化、卷 4 时间线在调用理论概念时有稳定参照。
- Phase 4（卷 1 物质基础）放第三，是因为物质纹理的确立要早于制度卷——卷 3 大量调用物质前提（聚变上线、Muule、积分卡的物理形态等）。
- Phase 5—7（卷 3 / 卷 4 / 卷 5）是工程最大的三块，按"制度 → 时间线 → 地理"顺序展开。
- Phase 8（卷 6）需要前面所有卷的把握度声明做基础，必须最后写。
- Phase 9（附录）随时可写零散填充，但正式版本在 Phase 9 集中校对。
- Phase 10 通稿是必备最后步骤。

## 四、用户介入点

以下时刻 AI 必须停下来等作者：

1. **Phase 1 完成后**：作者审定跨路径对比章设计稿和 5.7.4 拆分方案。
2. **Phase 2 完成后**（卷 0 导论）：作者通读卷 0，确认新书的"皮"是否合意。这是最重要的一次用户介入——卷 0 不通过则全书停。
3. **Phase 3 完成后**（卷 2 理论）：作者通读卷 2，确认理论部分的文体方向。
4. **Phase 4—7 各 Phase 完成后**（每一卷）：作者抽样阅读，给出"通过 / 部分返工 / 全部返工"的判定。
5. **遇到任何阻塞**（见自动恢复协议三）：作者裁定后续。

不必等作者的时刻：单章完成、Phase 内章节顺序、字数微调、史鉴选用——这些 AI 自决。

## 五、风险预案

**风险 R1：作者对卷 0 不满意。**
应对：卷 0 是新书的入口，不能含糊。如果作者通读后觉得方向偏，**整个 Phase 2 全部重写**，宁可推迟 Phase 3。这是设计上的"硬刹车点"。

**风险 R2：Phase 5 卷 3 路径纵切产生重复。**
应对：每条路径章 8 节左右，跨路径同质化的部分（如"配给/积分的计算逻辑"）容易重复。在 Phase 5 启动时先写一份"路径章共用部分"备忘录，明确各路径只写各自特色，共用机制只在卷 3 章一统一交代。

**风险 R3：Phase 6 卷 4 时间轴章节字数失控。**
应对：动荡期溶解后，章三/章四/章五成了大章。每章拆为 1 万字以内的可独立写的小段，章末做一次串联即可。

**风险 R4：Phase 8 卷 6 把握度处理不当。**
应对：卷 6 是全书最难的一卷——既要严格自识其限，又要给读者实质性的判断。建议在 Phase 8 启动前先与作者讨论"卷 6 的把握度示范片段"，确认调子再展开。

**风险 R5：跨会话术语漂移。**
应对：每完成一卷，把该卷出现的新术语回填到代号对照表 v2、v3。下一卷开写前先读最新版对照表。

**风险 R6：作者长时间未介入，工作积压。**
应对：每 Phase 结束就停下来等作者，不要"先把后续卷往前赶"。等待期间 AI 可做的事：补史鉴对照表的东亚史例库存（约 30—50 条）、补关键字词典、整理已写卷的索引。
