# 重组的关键决策点（待作者裁定）

## 1. 卷 3 是横切还是纵切？

v143 第四部分是**横切**（按制度子项：社保、财政、货币、监控分章，每章内对照各路径）。本草案改为**纵切**（每条路径独立成章，制度子项作为该章节的小节）。

**纵切优点：** 读者可以"专攻一条路径"读完整图景；每条路径的内在自洽更清晰。
**纵切缺点：** 跨路径横向比较需要在卷尾专门处理，否则读者不易看出"两条路径的同一项制度有什么不同"。
**折中方案：** 纵切为主，每章末加一节"与其他路径对比"。

**裁定项：** 是否采纳纵切？

## 2. 卷 4 是否拆出"动荡期"为独立卷？

当前 v143 第七部分是动荡期独立成部分。本草案把动荡期溶解进卷 4 的时间轴（章三第一次动荡 + 章四+章五补丁与交汇期）。

**溶解优点：** 时间轴整体性强；动荡期不再是孤立单元；读者按时间顺序经历"裸露期 → 补丁 → 动荡 → 稳态"的完整历程。
**溶解缺点：** 动荡期的政治分析失去专属舞台；台海冲突等热点章节被分散到不同小节。

**折中方案：** 溶解为主，但卷 4 章三末尾加一节"动荡期的整体特征：本卷读者笔记"作为专门收束。

**裁定项：** 是否采纳溶解？

## 3. 数据与媒体放卷 3 还是单独成卷？

当前草案：3.18 城市数字基础设施、3.19 数据基础设施、3.20 媒体生态、3.21 信息战 → 全部移入卷 3（制度分化）章二/三的"数据与媒体"小节 + 章九"城市数字基础设施与信息战"。

**入卷 3 优点：** 数据与媒体本质是制度技术，与各路径的具体制度形态紧密相关。

# 风险清单

**风险一　纵切与横切的论证连贯性。**
v143 现版第四部分按制度子项横切。某些跨路径的论证（如"美式路径用立法竞争和选举压力驱动，中国路径用行政指令和财政补贴驱动"）依赖于把两条路径放在同一节里对比。纵切后，这种对比要么失去，要么需要在卷尾汇总章重做。重做需要新写约 1—1.5 万字。

**风险二　锚点年切片与时间轴的互文管理。**
序·一的五个锚点年切片是新书的"皮"，写得稍弱整本书的入口都垮。但写得太厚又会与卷 4 重叠。需要严格的字数约束（卷 0 版每年 1500 字、卷 4 版每年 800 字）。

**风险三　卷 6 未来开放的把握度风险。**
此卷专门处理推演的不确定性。如果写得过于谨慎，会给人"作者自己也不信自己写的"的感觉；如果写得过于断言，违反推演纪律。文体上需要严格遵守把握度梯度规七。

**风险四　卷 3 第十二章（2072 年制度稳态）与卷 4 第六章（稳态浮现）的边界。**
两者都在描述 2072 年稳态。卷 3 第十二章侧重"是什么"（横截面），卷 4 第六章侧重"如何到达"（时间收束）。这个边界需要在写作时严格守护，否则会重复。

**风险五　v143 中某些章节的归属在草案中模糊。**
特别是：5.6.2 民主东亚（应入卷 3 还是卷 5？建议入卷 3 章四盟邦路径）；5.7.4 资源出口国 D 类（应入卷 3 章六还是卷 4 章七？建议入卷 3 章六）；4.13 国际金融（卷 3 章七，本草案已确定）。需要作者最终裁定。
