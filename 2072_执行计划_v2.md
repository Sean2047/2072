## 一、总则

**工程目标：** 把 v143（28.7 万字理论推演文件）改写为 v144（约 67—70 万字六卷可读著作）。
**预计工期：** 5—7 个月。**任务单位：** 章（chapter）。**总章数：** 约 82 个（v1 估 80 个 + 卷 3 新增 2 章）。
**核心约束：**
- 一次会话上下文有限，须以章为最小独立单位写作。
- 工具集（文风手册 v2 / 代号对照表 v2 / 史鉴对照表）已稳定，写作时按规调用。
- 任何阻塞或裁定问题写入进度看板，不自行拍板。

- 卷 0 v4（含作者前言稳重调子+ AI 协作明示）已通过审定。

**作者已裁定的结构决策（2026-05-08 至 09，共 11+1 项）：**
1. 卷 3 制度分化采用纵切（每条路径独立成章）；跨路径对比放卷 3 章十三。
2. 卷 4 把动荡期溶解进时间轴。
3. 数据与媒体按草案：部分入路径章、部分入卷 3 章十一。
4. 锚点年切片两版互文：卷 0 厚版（每年 1500—3000 字）、卷 4 简版（每年 800 字）。
5. V1—V8、A—F 代号部分下沉：正文用中文名，章末附对照表。
6. 5.6.2 民主东亚归卷 3 章四盟邦路径。
7. **5.7.4 资源出口国分拆：卷 3 讲过程、卷 5 讲结果**（双卷协同，已设计稿落地）。
8. 4.13 国际金融归卷 3 章七。
9. 卷 0 路径速览与卷 3 路径总图各司其职：卷 0 是 5 分钟轮廓，卷 3 是该路径全图。
10. **七条路径独立列出**（不合并附庸到中国路径、不合并资源国到遗忘地带）。
11. 文风稳重：去 false modesty，明示 AI 协作。
12. **卷 3 章数从 12 扩为 14**（保留七条路径的衍生影响）。

## 五、用户介入点

以下时刻 AI 必须停下来等作者：

1. **Phase 1 完成后** ✅
2. **Phase 2 完成后**（卷 0 导论） ✅
3. **Phase 3 完成后**（卷 2 理论）：作者通读卷 2，确认理论部分的文体方向。**这是工程第三个硬刹车点。**
4. **Phase 4—7 各 Phase 完成后**（每一卷）：作者抽样阅读，给出"通过 / 部分返工 / 全部返工"的判定。
5. **遇到任何阻塞**：作者裁定后续。

不必等作者的时刻：单章完成、Phase 内章节顺序、字数微调、史鉴选用——这些 AI 自决。

## 六、风险预案（沿用 v1，新增一条）

**新增风险 R7：稳重调子失守。**
v2 文风手册新增的"规十一稳重原则"是定调改动。后续 Phase 3—7 的写作中可能反弹回 false modesty 的旧习惯。应对：每章完成前自查规十一；每个 Phase 末由作者抽样检查"是否有不必要的'笔者无意夸大'、'本书雄心远不能与...'之类语句"。
