# 《2072》文内文献写作规范与首批篇目规划

**版本**：v1（2026-07-18，N-3）
**性质**：规范与规划文档，不入词条管线；正文撰写方式届时逐篇定（责任方N-1裁定为"AI（规范与规划）"，正文可AI起草或作者撰写，视篇目）。
**前置**：R2裁定文内文献与反对文集各自独立成区，`/documents/`落位（D-074）；挂载依赖N-1登记表机制（已就位）。

## 1　与反对文集的体裁区分

反对文集（`/objections/`）是**世界外**批评——现实读者视角评论理论框架本身。文内文献（`/documents/`）是**世界内**文本——2026–2076年间世界观内部实际产出的文书、读本、访谈、社论等，写作者要假装自己就是那个年代、那个身份的作者/受访者/发文机构，不评论"《2072》这份文件"（世界内根本不存在这份文件），只呈现该年代的知识状态与生活质感。两者体裁不可混用，此边界与N-2②禁止事项5互为镜像。

## 2　文献类型清单

| 类型 | 说明 | 典型路径 |
|---|---|---|
| 教材/读本节选 | K12或成人教育文本，体现官方/主流知识体系如何向公民讲述多态社会 | 各路径均可，A1/A2差异显著 |
| 制度用户手册 | BSC账户、供应票据等制度的操作层说明书，呈现制度的"界面"而非"设计意图" | A1（BSC）/A2（供应票据） |
| 官方公报/白皮书 | 政府或治理机构的正式文书，绩效叙事、政策解释 | 尤见于A2绩效合法性话语，A1亦有对应文体（政策说明） |
| 社论/评论文章 | 媒体（含自媒体）对时局的即时评论，是7.4"意识形态战争"主题的直接展品 | A1/A2措辞差异最大化的文体 |
| 口述史/访谈 | 事后回顾体，可跨年代（如2078年回顾2065年），承载代际交接窗口等长时段机制 | 不限路径 |
| 书信/私人文书 | 个人视角，尤适合呈现局限五（意义危机框架的文化局限）指向的非典型体验 | 尤适合E类/依附路径 |
| 公文/日志 | 具体事件或系统运行记录，可为事件层示例提供世界内旁证（不替代接口声明） | 不限路径 |

## 3　语域规范

### 3.1 世界内知识状态写作（核心纪律）

每篇文献的写作者必须严格代入其声称的时间点与身份的**知识状态**，不得携带"未来已展开"的上帝视角。具体要求：
- 若文献声称写于2032年，行文中不得出现只有2072年视角才会知道的因果结论（例如不能说"这正是后来相互忽视均衡的雏形"）；
- 文献中人物/机构对自身处境的理解应带有当时的局限、误判与情绪化判断的空间，不必（也不应）呈现理论正文式的机制清晰度；
- 文献是理论机制的**世界内证据碎片**，不是理论正文的转译——同一机制在文献里可以只呈现现象，不点破机制名称（不出现"V7""BSC""相互忽视均衡"等理论正文的分析代号，代号是分析框架的产物，世界内人物不这样谈论自己的生活，此纪律与N-4入口段禁用代号清单同源）。

### 3.2 A1/A2文本措辞差异（7.4节的具体展品）

路径叙事风格差异本身就是7.4"意识形态战争"主题的展品，文内文献是让这一命题从理论陈述变为可读证据的载体：

- **A1文本特征**：个人主义修辞（"你的选择""你的社区"）；多元观点竞争的痕迹（同一议题在文中留有争议空间，不给出唯一正确答案）；治理话语倾向程序性正当（"根据XX法案""经过XX听证"）；媒体/社论体裁尤其应呈现观点交锋而非统一口径。
- **A2文本特征**：集体主义修辞（"我们的家园""共同富裕的下一阶段"）；绩效合法性话语（用具体数字与成就叙事支撑正当性，而非程序正当）；官方口径统一，个体声音（如口述史）如涉及体制评价需体现克制或间接表达，而非直接批评式表达；用户手册/公报体裁尤其应呈现统一的、去争议化的说明语气。

两类措辞差异不是刻板印象化的简单对立标签，而应落到具体词汇选择、句式结构与论证方式上，写作时需回查7.1（技术-经济-制度互动机制）与7.4正文，确保措辞差异与理论正文已定义的机制一致。

### 3.3 与理论正文的接缝

文内文献不进入词条管线，但通过 `external_works.json` 登记表挂载指针（N-1机制），指向其呈现的机制所在节号或词条ID。每篇定稿后，登记表 `targets` 字段填入其世界内证据对应的理论节号（可多个），供网站在对应词条页/全文分节页渲染"相关文献"指针区。

## 4　《1984》致敬升格备考项——合并裁定

**原备考内容**："谁有权命名痛苦"作为贯穿主题的可选升格项（登记于历史日志，改与N-3合并裁定）。

**裁定**：**不设立为独立作品或独立主题工程**，而是作为**文内文献首批篇目的选题准则之一**折入本轮规划——理由：这一主题（对痛苦、失序、异常状态的命名权掌握在谁手中）与本文件7.4节"意识形态战争：叙事主战场"、3.20节"媒体生态"、1.2.1节V7痛苦政治化机制高度同构，用一个独立包装的"1984致敬"项目会与既有机制重复搭建一套平行叙事框架；折入文内文献首批篇目，则可以用A1/A2两种官方与半官方文本的并置，直接把"命名权"具体化为可读证据，且天然获得了3.1语域规范（3.2节A1/A2措辞差异）与登记表挂载机制的支持，不需要额外工程。

**落地方式**：首批篇目中的《绩效合法性白皮书节选》（A2官方公报体）与《动荡期社论：论组织门槛与沉默的多数》（A1社论体）两篇明确承载这一主题——前者展示"命名"如何服务于绩效合法性叙事（例如把某类抗议重新定义为"个别极端案例"或将"意义危机"官方定性为"发展阶段性现象"）；后者展示A1体系下"命名"如何在多元竞争的话语场中被公开争夺而非被统一裁定。两篇并置阅读即完成"谁有权命名痛苦"主题的呈现，不必另立标题或另立工程条目。此裁定待作者确认；若作者认为分量不足，可在后续批次追加第三篇（如E类/依附路径视角的"命名权缺席"对照）。

## 7　与登记表的挂载流程

1. 篇目定稿（状态：规划中→已发布前的中间态，可先以"规划中"登记标题与targets，正文完成后更新url与status）；
2. 登记入 `website_prototype/external_works.json`，`category` 填"文献"；
3. 运行 `check_entries.py` 校验targets可解析；
4. `build.py` 重建，确认指针渲染到对应词条页/分节页。

流程与反对文集共享同一登记表机制（N-1已落地），无需额外工程。
