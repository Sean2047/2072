## 一、卷 0 卷首页

### 卷 0　导论

> 一切真正的变化都先在语言里发生。

#### 本卷构成

- 作者前言
- 序·一　五个锚点年切片（2027 / 2037 / 2044 / 2052 / 2068）
- 序·二　七条路径速览（美式 / 中国 / 盟邦 / 附庸 / 持平 / 资源国 / 遗忘）

#### 本卷意图

卷 0 是本书的入口，**不承担任何论证任务**。它的功能只有两条：(一) 给读者一张地图，让读者在进入正文之前已经能用三句话向朋友介绍本书在讲什么；(二) 给读者一些工具——锚点年、路径名、术语、阅读法——让读者读到正文时不会因不熟悉术语而打断节奏。

如果读者觉得卷 0 内容稀薄、结构松散——那是有意的。**入门书页要稀薄，正文页才能厚实。** 读者诸君读完卷 0 进入卷 1 时若觉得"原来本书的核心内容在这里才开始"，那卷 0 就完成了它的工作。

## 二、序·一与序·二之间的衔接段

> 此段位于序·一·5（2068 稳态之年）末尾"是为 2068 年"之后，序·二开篇之前。

五个锚点年勾勒出从动荡期开端到新稳态成熟的弧线——时间维度上的全图。但 2072 年的世界不是一根直线，而是一组并行的国家路径。同一物理年份里，不同的国家走在不同的历史阶段——美式路径在 2052 年完成联邦立法之时，遗忘地带尚处于结构性裸露期；中国路径的票据制成熟形态早于美式路径的积分托底制；盟邦路径的福利再校准节奏介于两者之间；附庸路径的制度形式由中国路径输入但执行能力受限；持平国家走在自主双轨的另一条路径上；资源国正在经历内部分化最剧烈的过程。**时间是一根，但走时间的脚不止一双。**

为帮助读者理解这种"同物理时间、不同历史阶段"的并存现象，下面给出七条路径的速览。每条约一千字，含定位 / 代表国家 / 政治经济逻辑 / 社会形态 / 主要风险五项。读完七条路径速览，读者就拿到了本书后续诸卷会反复回到的两根坐标：时间纵轴 + 路径横轴。

## 五、卷 0 末尾向卷 1 的衔接段

> 此段位于序·四末"附录 F 版本史 + 二次创作开放许可"之后，作为卷 0 的整体收束，引向卷 1 物质基础。

至此卷 0 完结。

下一卷是卷 1 物质基础。

为什么是物质基础先行？为什么不先讲理论？

理由有两条。**第一，理论需要支点。** 抽象的论述若没有具体的物质纹理作支点，读者很容易在阅读中迷失——读到"劳动价值的结构性弱化"时若没有先看见过 2072 年的工厂、Muule、合成燃油站、积分卡，这一论述就只是一组抽象命题，难以在脑中形成具体形象。卷 1 物质基础就是为了先给读者建立这种形象。

**第二，物质本身是论证的一部分。** 本书一以贯之的立场是：制度形态由物质条件决定，不是凭空设计出来的。AI 与聚变这两条物质主轴，在很大程度上决定了七条路径在结构上的可行解集合。理论解释了"为什么"，物质决定了"是什么"——两者缺一不可，但顺序上物质先行。

卷 1 物质基础将分八章展开：两条主轴（AI 与聚变）、能源新地理、全自动制造与原料、医疗与生命、人与机器人与机器、通信量子与基础网络、物件与日常、移动与连通。这是 2072 年世界的物质骨架——读完之后，本书后续所有的制度分化、地区差异、动荡与稳定的论述，都将在这副骨架上展开。
